# QuorielDB
Интуитивно понятный и высокопроизводительный интерфейс для работы с базами данных в ForgeScript, обеспечивающий надежное и масштабируемое хранение данных с помощью LMDB.

## Установка
```
npm i https://github.com/quoriel/db.git lmdb
```

## Подключение
```js
const { ForgeClient } = require("@tryforge/forgescript");
const { QuorielDB } = require("@quoriel/db");

const client = new ForgeClient({
    extensions: [
        new QuorielDB()
    ]
});

client.login("...");
```

## Функции
Перечень всех доступных функций можно найти в директории src/functions. Ознакомьтесь с каждой для лучшего понимания возможностей дополнения.

## Значения по умолчанию
Для того чтобы установить значения по умолчанию для переменной, в корневой папке вашего бота будет создана папка **quoriel/db**. Внутри этой папки вы найдете файл **variables.json**, который будет содержать все значения по умолчанию.

Этот файл позволяет задать начальные значения для переменных, которые могут быть использованы, если функция **$getVar** не вернет результата. Вы можете отредактировать файл **variables.json** для настройки значений по умолчанию под ваши нужды.
